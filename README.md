# Wednesday-assignment

## 環境構築

```
git clone git@github.com:Itsuki0016/Wednesday-assignment.git
```


ホームディレクトリで以下を実行しましょう。

```
screen 
```

screenが入ってない人は以下を参照
[screen install ] (https://qiita.com/Chi_corp_123/items/db23df2b6510533b6d43)


ctrl+A >> C でスクリーンを2つ作成し片方で

```
docker compose up 
```

もう片方は作業用に使いましょう。
